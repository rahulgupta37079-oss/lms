22. **Session 22**: PROJECT: Light-Up Card

#### **Sessions 25-36: Interactive Play**

34. **Session 34**: PROJECT: Dancing Robot

36. **Session 36**: PROJECT: Game Robot

#### **Sessions 37-48: Final Projects**
37. **Session 37**: Project Planning (My Robot Idea)
38. **Session 38**: Gathering Materials
39. **Session 39**: Building Phase 1 (Structure)
40. **Session 40**: Building Phase 2 (Mechanics)
