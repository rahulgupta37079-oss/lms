## üìù **Session Structure (60 Minutes)**

### **Every Session Includes:**

1. **Introduction (5 mins)**
   - Recap previous session
   - Introduce today's topic
   - Show learning objectives

2. **Video Lecture (10-15 mins)**
   - Concept explanation
   - Component demonstration
   - Real-world examples

3. **Interactive Learning (15-20 mins)**
   - Theory with visuals
   - 3D component models
   - Interactive simulations
   - Quick quiz (5-7 MCQs)

4. **Hands-on Activity (25-30 mins)**
   - Step-by-step instructions
   - Build/code activity
   - Test and troubleshoot
   - Teacher guidance

5. **Wrap-up & Assessment (5-10 mins)**
   - Review what was learned
   - Photo submission (for projects)
   - Reflection questions
   - Preview next session
